# Swiggy Restaurants Data Analysis

## 📌 Description
This project analyzes the Swiggy Restaurants Dataset containing over 140,000 restaurant records across India. The analysis focuses on cuisines, locations, ratings, pricing, offers, and vegetarian options to identify food trends and customer preferences.

### 📊 Key Insights
- Most popular cuisines: North Indian, Chinese, and Indian
- Cities with high restaurant density: Hyderabad, Kanpur, Indore, Central Goa, and Kanchrapara
- Frequently listed chains: KFC, Domino’s Pizza, and Pizza Hut
- Most restaurants had an average cost for two between ₹100–₹300
- Many restaurants provided discounts and promotional offers
- Dataset included both vegetarian and non-vegetarian restaurant information
